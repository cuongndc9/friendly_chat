# friendly_chat

🙌👻 Building a simple chat with Flutter.

## Quick start

- Get packages from **pub** for Flutter app:

```bash
flutter packages get
```

- Run project:

```bash
flutter run
```

- Login with account:

```
username: admin
password: admin
```
